# 🎓 Student Dropout Analysis Dashboard

## 📌 Project Overview

The **Student Dropout Analysis Dashboard** is a data science project that analyzes student academic data and visualizes patterns related to study habits and performance.
The project uses **Python, Pandas, Matplotlib, and Streamlit** to create an interactive dashboard that helps understand factors affecting student performance.

---

## 🎯 Objectives

* Analyze student academic data
* Visualize key factors such as **study time and absences**
* Build an **interactive dashboard**
* Provide simple performance predictions

---

## 🛠️ Technologies Used

* **Python**
* **Pandas** – Data manipulation
* **Matplotlib** – Data visualization
* **Streamlit** – Interactive dashboard
* **Scikit-learn** – Machine learning models

---

## 📂 Dataset

The dataset used in this project is the **Student Performance Dataset**.

Dataset file:
student-mat.csv

It contains features such as:

* Study time
* Absences
* Parental education
* Alcohol consumption
* Previous grades
* Student performance indicators

---

## 📊 Dashboard Features

The Streamlit dashboard includes:

### 1️⃣ Dataset Preview

Displays the first few rows of the dataset.

### 2️⃣ Dataset Information

Shows dataset size and structure.

### 3️⃣ Study Time Analysis

Bar chart showing distribution of study hours among students.

### 4️⃣ Absence Analysis

Histogram showing distribution of student absences.

### 5️⃣ Simple Performance Prediction

Users can input:

* Study time
* Number of absences

The dashboard predicts whether a student is **likely to perform well or be at risk**.

## ▶️ How to Run the Project

### 1️⃣ Install Required Libraries

```bash
pip install pandas matplotlib streamlit scikit-learn
```

### 2️⃣ Run the Streamlit App

```bash
streamlit run app1.py
```

The dashboard will open in your browser.
